# 💡 백준 1193번: 분수찾기

## 문제 설명
X번째 분수를 찾는 문제입니다.  
분수는 다음과 같은 순서로 배열됩니다.

1/1
1/2, 2/1
3/1, 2/2, 1/3
1/4, 2/3, 3/2, 4/1
5/1, 4/2, 3/3, 2/4, 1/5
...



- **짝수 번째 대각선**: 분모가 1씩 증가, 분자가 1씩 감소  
- **홀수 번째 대각선**: 분자가 1씩 증가, 분모가 1씩 감소  

## ✍ 알고리즘 설명
1. `n`을 1부터 증가시키면서 `x`에서 `n`을 빼면서 `x`가 몇 번째 대각선에 속하는지 찾음.
2. `n`이 **짝수**면 `(x/n-x+1)`, **홀수**면 `(n-x+1/x)`을 구함.

## 🛠 코드

```python
x = int(input())  
n = 1  # 대각선 수

while x > n:  
    x -= n
    n += 1

a, b = 0, 0  # 분자, 분모
if n % 2 == 0:
    a = x
    b = n - x + 1
else:
    a = n - x + 1
    b = x

print(f"{a}/{b}")
